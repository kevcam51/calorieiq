## How to read availability
- `.com` for pronounceable short names is **essentially all registered** — but many are **parked by
  brokers** (afternic / atom / brandbucket / sedo / spaceship), meaning **no company uses the name** (the
  name can still be trademark-clear; the domain just costs money). "Registered" ≠ "a business owns it."
- Practical path for an app: **`.app` / `.fit` / `.co`** — often free for the exact short name.
- A free domain does NOT guarantee no company uses the name → always web-check the name too. For launch,
  a **USPTO trademark search** (or a lawyer) is the real clearance; web check is a strong signal only.

## Verdict on names checked so far — ALL `.com`s taken
- **Slyde / Vyde** — heavily used (Slyde = several apps + a watch brand; Vyde = adtech + a US accounting firm).
- **Onvo** (Nio EV brand), **Kova** (KOVA Corp + software), **Movi** (many), **Kasa** (TP-Link, huge),
  **Zuma** (restaurants/game/nutrition), **Ovi** (Nokia/Ovia) — all taken, several big.

  payments; a property firm exists), **Kruze** (≈ Kruze Consulting), **Flowvra** (≈ Flowva/Flova/Flowra).
